#1
Two Sum
Array Hash Map
Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.
Input
nums = [2,7,11,15], target = 9
Output
[0, 1]
Key insight: For every number x, check if target - x already exists in a hash map. One pass — O(n) vs O(n²) brute force.
Python 
def twoSum(nums: list[int], target: int) -> list[int]:
    seen = {}  # value → index
    for i, n in enumerate(nums):
        complement = target - n
        if complement in seen:
            return [seen[complement], i]
        seen[n] = i
    return []
Time: O(n) Space: O(n)
🔄 Follow-up

What if the array is sorted? Use two pointers (l, r). Move l right if sum < target, r left if sum > target. O(n) time, O(1) space. LeetCode 167.

What if you need all pairs? Collect all matching index pairs into a list. Same O(n) approach but append instead of return.

Contains Duplicate
Array Hash Set
Given an integer array nums, return true if any value appears at least twice in the array, and return false if every element is distinct.
Input
nums = [1, 2, 3, 1]
Output
True
Key insight: A set stores only unique values. If len(set(nums)) != len(nums), there's a duplicate. Or stream through and short-circuit early.
Python 
def containsDuplicate(nums: list[int]) -> bool:
    seen = set()
    for n in nums:
        if n in seen:
            return True
        seen.add(n)
    return False

# One-liner (but reads entire array first)
def containsDuplicate_v2(nums: list[int]) -> bool:
    return len(nums) != len(set(nums))
Time: O(n) Space: O(n)
🔄 Follow-up

Contains Duplicate II (LC 219): Duplicate within k distance? Use a sliding window set of size k. Add new, remove old (index i-k), check if already in set.

#4
Best Time to Buy and Sell Stock
Array Greedy
You are given an array prices where prices[i] is the price of a given stock on the ith day. Find the maximum profit you can achieve by buying one day and selling in the future.
Input
prices = [7,1,5,3,6,4]
Output
5 (buy at 1, sell at 6)
Key insight: Track the minimum price seen so far as you scan. At each step, the best profit you can make selling today is price - min_so_far. Update both greedily.
Python 
def maxProfit(prices: list[int]) -> int:
    min_price = float('inf')
    max_profit = 0
    for price in prices:
        if price < min_price:
            min_price = price
        elif price - min_price > max_profit:
            max_profit = price - min_price
    return max_profit
Time: O(n) Space: O(1)
🔄 Follow-up

Multiple transactions (LC 122): Add up every positive difference between consecutive days — sum(max(0, prices[i]-prices[i-1]) for i in range(1, n)).

At most 2 transactions (LC 123): DP with state machine — hold/sold states per transaction count.

#5
Majority Element
Array Greedy
Given an array nums of size n, return the majority element. The majority element is the element that appears more than ⌊n / 2⌋ times.
Input
nums = [3, 2, 3]
Output
3
Key insight: Boyer-Moore Voting Algorithm. Maintain a candidate and count. When count hits 0, switch candidate. The majority element always survives because it appears > n/2 times — it can "absorb" all opposition.
Python 
def majorityElement(nums: list[int]) -> int:
    candidate, count = None, 0
    for n in nums:
        if count == 0:
            candidate = n
        count += 1 if n == candidate else -1
    return candidate
Time: O(n) Space: O(1)
🔄 Follow-up

What if majority is > n/3 (LC 229)? Track two candidates simultaneously. Same idea, two slots. Final answer: verify both candidates actually exceed n/3 count.

No guaranteed majority? Add a verification pass after finding the candidate.

#25
Squares of a Sorted Array
Array Two Pointers
Given an integer array nums sorted in non-decreasing order, return an array of the squares of each number sorted in non-decreasing order.
Input
nums = [-4, -1, 0, 3, 10]
Output
[0, 1, 9, 16, 100]
Key insight: Largest squares come from either end (most negative or most positive). Two pointers from both ends, fill result array from back to front. O(n) — no sort needed after squaring.
Python 
def sortedSquares(nums: list[int]) -> list[int]:
    n = len(nums)
    result = [0] * n
    l, r = 0, n - 1
    pos = n - 1   # fill from the back
    while l <= r:
        sq_l = nums[l] ** 2
        sq_r = nums[r] ** 2
        if sq_l > sq_r:
            result[pos] = sq_l; l += 1
        else:
            result[pos] = sq_r; r -= 1
        pos -= 1
    return result
Time: O(n) Space: O(n)
🔄 Follow-up

Why not just square then sort? That's O(n log n). This two-pointer approach exploits the sorted property for O(n). Always mention both to show you understand the trade-off.

#8
Move Zeroes
Array Two Pointers
Given an integer array nums, move all 0's to the end of it while maintaining the relative order of the non-zero elements.
Input
nums = [0,1,0,3,12]
Output
[1,3,12,0,0]
Key insight: Two-pointer approach - one pointer for placement position, another for scanning. When you find a non-zero, swap it with the placement pointer position. O(n) time, O(1) space.
Python 
def moveZeroes(nums: list[int]) -> None:
    place = 0  # position to place next non-zero
    for i in range(len(nums)):
        if nums[i] != 0:
            nums[place], nums[i] = nums[i], nums[place]
            place += 1
Time: O(n) Space: O(1)
🔄 Follow-up

What if you need to minimize writes? Count zeros first, then shift non-zeros left and fill end with zeros. Fewer swaps but same O(n) time.

Ransom Note
String Hash Map
Given two strings ransomNote and magazine, return true if ransomNote can be constructed by using the letters from magazine and false otherwise.
Input
ransomNote = "aa", magazine = "aab"
Output
True
Key insight: Count each character in magazine. For each character needed by ransomNote, decrement. If any count goes negative, return False.
Python 
from collections import Counter

def canConstruct(ransomNote: str, magazine: str) -> bool:
    mag_count = Counter(magazine)
    for ch in ransomNote:
        mag_count[ch] -= 1
        if mag_count[ch] < 0:
            return False
    return True

# One-liner using Counter subtraction:
def canConstruct_v2(ransomNote: str, magazine: str) -> bool:
    return not (Counter(ransomNote) - Counter(magazine))

#7
Valid Palindrome
String Two Pointers
Given a string s, return true if it is a palindrome, considering only alphanumeric characters and ignoring cases.
Input
s = "A man, a plan, a canal: Panama"
Output
True
Key insight: Two pointers from both ends, skipping non-alphanumeric characters. Compare lowercased. No need to build a cleaned string first — do it in-place with pointer movement.
Python 
def isPalindrome(s: str) -> bool:
    l, r = 0, len(s) - 1
    while l < r:
        while l < r and not s[l].isalnum(): l += 1
        while l < r and not s[r].isalnum(): r -= 1
        if s[l].lower() != s[r].lower(): return False
        l += 1; r -= 1
    return True
Time: O(n) Space: O(1)
🔄 Follow-up

Valid Palindrome II (LC 680): Can delete at most one character? When mismatch found, try skipping left or right character and check if remaining is palindrome.

#9
Add Binary
String Bit Math
Given two binary strings a and b, return their sum as a binary string.
Input
a = "11", b = "1"
Output
"100"
Key insight: Simulate grade-school binary addition from right to left. Carry propagates just like decimal but with modulo 2. Build result in reverse, then flip at the end.
Python 
def addBinary(a: str, b: str) -> str:
    i, j = len(a) - 1, len(b) - 1
    carry = 0
    result = []
    while i >= 0 or j >= 0 or carry:
        total = carry
        if i >= 0: total += int(a[i]); i -= 1
        if j >= 0: total += int(b[j]); j -= 1
        result.append(str(total % 2))
        carry = total // 2
    return ''.join(reversed(result))
Time: O(max(m,n)) Space: O(max(m,n))
🔄 Follow-up

Add strings (decimal) (LC 415): Same logic, change modulo to 10. Interviewer may ask about very large numbers — this approach handles arbitrary precision.

Valid Anagram
String Hash Map
Given two strings s and t, return true if t is an anagram of s, and false otherwise.
Input
s = "anagram", t = "nagaram"
Output
True
Key insight: Two strings are anagrams iff their character frequency maps are identical. Counter makes this a one-liner. Sorting works too but is O(n log n).
Python 
from collections import Counter

def isAnagram(s: str, t: str) -> bool:
    return Counter(s) == Counter(t)

# Manual (no Counter) — O(1) space for 26 lowercase letters:
def isAnagram_v2(s: str, t: str) -> bool:
    if len(s) != len(t): return False
    freq = [0] * 26
    for a, b in zip(s, t):
        freq[ord(a) - ord('a')] += 1
        freq[ord(b) - ord('a')] -= 1
    return all(f == 0 for f in freq)
Time: O(n) Space: O(1) — fixed 26 chars
🔄 Follow-up

Unicode characters (not just lowercase)? Use a dict / Counter — can't use fixed array anymore. O(n) space.

#8
Longest Palindrome
String Greedy
Given a string s, return the length of the longest palindrome that can be built with those letters. Letters are case sensitive.
Input
s = "abccccdd"
Output
7 (e.g., "dccaccd")
Key insight: Count characters. For each character, use count // 2 * 2 for the palindrome. If any odd count exists, add 1 for a single middle character.
Python
def longestPalindrome(s: str) -> int:
    res, odd = 0, False
    for count in collections.Counter(s).values():
        res += count // 2 * 2
        if count % 2 == 1: odd = True
    return res + 1 if odd else res
Time: O(n) Space: O(k) - charset size
#26
Longest Common Prefix
String
Write a function to find the longest common prefix string amongst an array of strings. If there is no common prefix, return an empty string "".
Input
strs = ["flower","flow","flight"]
Output
"fl"
Key insight: Sort the array — the common prefix of the lexicographically smallest and largest strings is the common prefix of all strings. Only two comparisons needed after sorting.
Python 
def longestCommonPrefix(strs: list[str]) -> str:
    if not strs: return ""
    strs.sort()
    first, last = strs[0], strs[-1]
    i = 0
    while i < len(first) and i < len(last) and first[i] == last[i]:
        i += 1
    return first[:i]

# Alternative: vertical scan (no sort)
def longestCommonPrefix_v2(strs: list[str]) -> str:
    if not strs: return ""
    for i, ch in enumerate(strs[0]):
        if any(i >= len(s) or s[i] != ch for s in strs[1:]):
            return strs[0][:i]
    return strs[0]
Time: O(n log n + m) sort / O(nm) vertical Space: O(1)
🔄 Follow-up

Large number of strings? Binary search on prefix length: check if prefix of length mid is common to all. O(nm log m).

#27
Roman to Integer
String Hash Map
Roman numerals are represented by seven different symbols: I, V, X, L, C, D and M. Given a roman numeral, convert it to an integer.
Input
s = "MCMXCIV"
Output
1994
Key insight: In Roman numerals, a smaller value before a larger one means subtraction (IV = 4). Scan right-to-left: if current value is less than the previous, subtract it; otherwise add it.
Python 
def romanToInt(s: str) -> int:
    val = {'I':1,'V':5,'X':10,'L':50,
           'C':100,'D':500,'M':1000}
    result = 0
    prev = 0
    for ch in reversed(s):
        curr = val[ch]
        result += curr if curr >= prev else -curr
        prev = curr
    return result
Time: O(n) Space: O(1)
🔄 Follow-up

Integer to Roman (LC 12)? Greedily subtract the largest Roman value that fits, appending its symbol. Use a list of (value, symbol) pairs in descending order.

#28
Palindrome Number
Math
Given an integer x, return true if x is a palindrome, and false otherwise.
Input
x = 121
Output
True
Key insight: Reverse only the second half of the number. Compare with the first half. Stops at the midpoint — no overflow risk, no string conversion needed. Negative numbers are never palindromes.
Python 
def isPalindrome(x: int) -> bool:
    if x < 0 or (x % 10 == 0 and x != 0): return False
    rev = 0
    while x > rev:              # stop at midpoint
        rev = rev * 10 + x % 10
        x //= 10
    return x == rev or x == rev // 10  # even/odd length

# String approach (simpler, slightly less optimal)
def isPalindrome_str(x: int) -> bool:
    s = str(x)
    return s == s[::-1]
Time: O(log n) Space: O(1)
🔄 Follow-up

Without converting to string? The half-reversal approach above is exactly that. Key edge case: numbers ending in 0 (except 0 itself) can never be palindromes.

#10
Merge Two Sorted Lists
Linked List Two Pointers
You are given the heads of two sorted linked lists list1 and list2. Merge the two lists in a one sorted list. The list should be made by splicing together the nodes of the first two lists.
Input
list1 = [1,2,4], list2 = [1,3,4]
Output
[1,1,2,3,4,4]
Key insight: Dummy head node eliminates the edge case of "what's the head?". Then greedily pick the smaller node. Clean, no allocation beyond the dummy.
Python 
def mergeTwoLists(l1, l2):
    dummy = curr = ListNode(0)
    while l1 and l2:
        if l1.val <= l2.val:
            curr.next = l1; l1 = l1.next
        else:
            curr.next = l2; l2 = l2.next
        curr = curr.next
    curr.next = l1 if l1 else l2  # attach remaining
    return dummy.next
Time: O(m+n) Space: O(1)
🔄 Follow-up

Merge K sorted lists (LC 23): Use a min-heap. Push the head of each list. Pop the min, push its next. O(N log K) where K = number of lists.

#11
Linked List Cycle
Linked List Fast/Slow Ptr
Given head, the head of a linked list, determine if the linked list has a cycle in it. There is a cycle in a linked list if there is some node in the list that can be reached again by continuously following the next pointer.
Input
head = [3,2,0,-4], pos = 1 (cycle)
Output
True
Key insight: Floyd's Tortoise & Hare. Slow moves 1 step, fast moves 2. If there's a cycle, fast will lap slow — they'll meet. If fast hits null, no cycle.
Python 
def hasCycle(head) -> bool:
    slow, fast = head, head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow is fast:
            return True
    return False
Time: O(n) Space: O(1)
🔄 Follow-up

Find where the cycle starts (LC 142): After slow==fast, reset one pointer to head. Move both one step at a time — they meet at the cycle entry. Mathematical proof: distance from head to entry = distance from meeting point to entry.

#12
Reverse Linked List
Linked List
Given the head of a singly linked list, reverse the list, and return the reversed list.
Input
head = [1, 2, 3, 4, 5]
Output
[5, 4, 3, 2, 1]
Key insight: Three-pointer dance: save next before overwriting, flip the arrow, advance. The assignment order matters — save next_node first or you lose the rest of the list.
Python 
# Iterative — O(1) space (preferred in interviews)
def reverseList(head):
    prev, curr = None, head
    while curr:
        next_node = curr.next   # save before overwrite!
        curr.next = prev
        prev = curr
        curr = next_node
    return prev

# Recursive — O(n) call stack space
def reverseList_rec(head):
    if not head or not head.next: return head
    new_head = reverseList_rec(head.next)
    head.next.next = head
    head.next = None
    return new_head
Time: O(n) Space: O(1) iterative / O(n) recursive
🔄 Follow-up

Reverse in range [left, right] (LC 92): Find the node just before left, reverse the sublist, reconnect the ends.

Palindrome Linked List (LC 234): Find midpoint, reverse second half, compare both halves, restore (optional).

#30
Palindrome Linked List
Linked List Two Pointers
Given the head of a singly linked list, return true if it is a palindrome or false otherwise.
Input
head = [1, 2, 2, 1]
Output
True
Key insight: Three steps: (1) find midpoint with fast/slow, (2) reverse the second half in-place, (3) compare both halves. O(n) time, O(1) space. Optionally restore the list after.
Python 
def isPalindrome(head) -> bool:
    # Step 1: find middle
    slow = fast = head
    while fast and fast.next:
        slow = slow.next; fast = fast.next.next

    # Step 2: reverse second half
    prev = None
    while slow:
        slow.next, prev, slow = prev, slow, slow.next

    # Step 3: compare
    left, right = head, prev
    while right:
        if left.val != right.val: return False
        left = left.next; right = right.next
    return True
Time: O(n) Space: O(1)
🔄 Follow-up

Restore the list after check? Reverse the second half again after comparison. Same reversal code.

O(n) space approach? Copy values to array and use two-pointer check. Much simpler but uses O(n) space.

#13
Invert Binary Tree
Tree DFS
Given the root of a binary tree, invert the tree, and return its root.
Input
root = [4,2,7,1,3,6,9]
Output
[4,7,2,9,6,3,1]
Key insight: Post-order recursion: swap left/right children at every node. The base case (null) just returns null. Trust the recursion — after the recursive calls return, both subtrees are already inverted.
Python 
def invertTree(root):
    if not root: return None
    root.left, root.right = invertTree(root.right), invertTree(root.left)
    return root

# Iterative BFS version:
from collections import deque
def invertTree_bfs(root):
    if not root: return None
    q = deque([root])
    while q:
        node = q.popleft()
        node.left, node.right = node.right, node.left
        if node.left:  q.append(node.left)
        if node.right: q.append(node.right)
    return root
Time: O(n) Space: O(h) — h = height
🔄 Follow-up

What's the difference between BFS and DFS here? Both work. DFS uses O(h) stack space (skewed tree = O(n)), BFS uses O(w) queue space (w = max width). For balanced trees, BFS uses more memory.

#14
Maximum Depth of Binary Tree
Tree DFS
Given the root of a binary tree, return its maximum depth. A binary tree's maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node.
Input
root = [3, 9, 20, null, null, 15, 7]
Output
3
Key insight: The depth of a node is 1 + max(depth of left child, depth of right child). Base case: null node has depth 0. Pure bottom-up recursion — the simplest tree DFS pattern.
Python 
def maxDepth(root) -> int:
    if not root: return 0
    return 1 + max(maxDepth(root.left), maxDepth(root.right))

# BFS (iterative) — counts levels:
from collections import deque
def maxDepth_bfs(root) -> int:
    if not root: return 0
    q, depth = deque([root]), 0
    while q:
        depth += 1
        for _ in range(len(q)):
            node = q.popleft()
            if node.left:  q.append(node.left)
            if node.right: q.append(node.right)
    return depth
Time: O(n) Space: O(h)
🔄 Follow-up

Minimum depth (LC 111): Can't just use min() — a node with only one child should not count the null child as depth 1. Need to handle the case where left or right is null explicitly.

#15
Balanced Binary Tree
Tree DFS
Given a binary tree, determine if it is height-balanced. A height-balanced binary tree is defined as a binary tree in which the left and right subtrees of every node differ in height by no more than 1.
Input
root = [3,9,20,null,null,15,7]
Output
True
Key insight: Naive: check height at every node — O(n²). Better: return -1 as a sentinel from the DFS if any subtree is unbalanced, propagating failure upward. One O(n) pass.
Python 
def isBalanced(root) -> bool:
    def check_height(node) -> int:
        # Returns height, or -1 if unbalanced
        if not node: return 0
        left  = check_height(node.left)
        right = check_height(node.right)
        if left == -1 or right == -1: return -1
        if abs(left - right) > 1:  return -1
        return 1 + max(left, right)
    return check_height(root) != -1
Time: O(n) Space: O(h)
🔄 Follow-up

Convert unbalanced to balanced (LC 1382): Inorder traversal to get sorted array, then build balanced BST using divide and conquer (midpoint as root).

#16
Diameter of Binary Tree
Tree DFS
Given the root of a binary tree, return the length of the diameter of the tree. The diameter of a binary tree is the length of the longest path between any two nodes in a tree. This path may or may not pass through the root.
Input
root = [1, 2, 3, 4, 5]
Output
3 (path: 4→2→1→3 or 5→2→1→3)
Key insight: The diameter through any node = left_depth + right_depth (in edges). DFS returns height upward, but updates a global max at every node using left + right. The longest path doesn't have to pass through the root.
Python 
def diameterOfBinaryTree(root) -> int:
    diameter = 0

    def depth(node) -> int:
        nonlocal diameter
        if not node: return 0
        left  = depth(node.left)
        right = depth(node.right)
        diameter = max(diameter, left + right)
        return 1 + max(left, right)

    depth(root)
    return diameter
Time: O(n) Space: O(h)
🔄 Follow-up

Binary Tree Maximum Path Sum (LC 124): Same pattern! Instead of depths, track path sums. Allow negative subtrees to be ignored with max(dfs(node), 0). The global max updates with left + right + node.val.

#31
Lowest Common Ancestor of a BST
BST DFS
Given a binary search tree (BST), find the lowest common ancestor (LCA) node of two given nodes in the BST.
Input
root=[6,2,8,...], p=2, q=8
Output
Node 6 (root is LCA)
Key insight: BST property does all the work. If both p and q are smaller than root, LCA is in left subtree. If both larger, right subtree. Otherwise, root is the split point — it's the LCA.
Python 
def lowestCommonAncestor(root, p, q):
    while root:
        if p.val < root.val and q.val < root.val:
            root = root.left    # both in left subtree
        elif p.val > root.val and q.val > root.val:
            root = root.right   # both in right subtree
        else:
            return root         # split point = LCA

# Recursive (same logic, cleaner)
def lowestCommonAncestor_rec(root, p, q):
    if p.val < root.val and q.val < root.val:
        return lowestCommonAncestor_rec(root.left, p, q)
    if p.val > root.val and q.val > root.val:
        return lowestCommonAncestor_rec(root.right, p, q)
    return root
Time: O(h) — h = height Space: O(1) iterative
🔄 Follow-up

LCA in a regular binary tree (LC 236)? Can't use BST property. DFS: return node if it matches p or q. If left and right both return non-null, current node is LCA.

#32
Same Tree
Tree DFS
Given the roots of two binary trees p and q, check if they are the same or not. Two binary trees are considered the same if they are structurally identical, and the nodes have the same value.
Input
p = [1,2,3], q = [1,2,3]
Output
True
Key insight: Three cases at every node: (1) both null → same, (2) one null → different, (3) both exist → values must match AND both subtrees must be same recursively.
Python 
def isSameTree(p, q) -> bool:
    if not p and not q: return True   # both null
    if not p or  not q: return False  # one null
    return (p.val == q.val and
            isSameTree(p.left, q.left) and
            isSameTree(p.right, q.right))
Time: O(n) Space: O(h)
🔄 Follow-up

Subtree of Another Tree (LC 572)? At each node of the larger tree, call isSameTree. If it returns true, done. O(m×n) worst case. Can optimize with tree hashing.

#33
Symmetric Tree
Tree DFS
Given the root of a binary tree, check whether it is a mirror of itself (i.e., symmetric around its center).
Input
root = [1, 2, 2, 3, 4, 4, 3]
Output
True
Key insight: A tree is symmetric if the left subtree is a mirror of the right subtree. Two nodes are mirrors if their values match AND left.left mirrors right.right AND left.right mirrors right.left.
Python 
def isSymmetric(root) -> bool:
    def mirror(left, right) -> bool:
        if not left and not right: return True
        if not left or  not right: return False
        return (left.val == right.val and
                mirror(left.left,  right.right) and
                mirror(left.right, right.left))
    return mirror(root.left, root.right)
Time: O(n) Space: O(h)
🔄 Follow-up

Iterative with a queue? Push (left, right) pairs. Each iteration: dequeue a pair, check values, enqueue (l.left, r.right) and (l.right, r.left). O(n) space.

#34
Subtree of Another Tree
Tree DFS
Given the roots of two binary trees root and subRoot, return true if there is a subtree of root with the same structure and node values of subRoot.
Input
root=[3,4,5,1,2], subRoot=[4,1,2]
Output
True
Key insight: At every node of root, check if the subtree rooted there is identical to subRoot (reuse isSameTree). If the values differ, recurse on both children. Early termination when subRoot is null.
Python 
def isSubtree(root, subRoot) -> bool:
    def same(a, b) -> bool:
        if not a and not b: return True
        if not a or  not b: return False
        return a.val == b.val and same(a.left, b.left) and same(a.right, b.right)

    if not root: return False
    if same(root, subRoot): return True
    return isSubtree(root.left, subRoot) or isSubtree(root.right, subRoot)
Time: O(m×n) Space: O(h)
🔄 Follow-up

O(m+n) solution? Serialize both trees to strings (with delimiters to avoid partial matches), then check if subRoot's serialization is a substring of root's. Use KMP for O(m+n).

#35
Convert Sorted Array to BST
BST Divide & Conquer
Given an integer array nums where the elements are sorted in ascending order, convert it to a height-balanced binary search tree.
Input
nums = [-10, -3, 0, 5, 9]
Output
Height-balanced BST rooted at 0
Key insight: To get a height-balanced BST, always pick the midpoint as root. This ensures equal-length left and right subtrees. Recursively do the same for each half — classic divide and conquer.
Python 
def sortedArrayToBST(nums: list[int]):
    def build(lo, hi):
        if lo > hi: return None
        mid = (lo + hi) // 2
        node = TreeNode(nums[mid])
        node.left  = build(lo, mid - 1)
        node.right = build(mid + 1, hi)
        return node
    return build(0, len(nums) - 1)
Time: O(n) Space: O(log n) — recursion
🔄 Follow-up

Multiple valid answers? Yes — choosing (lo+hi)//2 vs (lo+hi+1)//2 as midpoint gives different valid BSTs. Both are height-balanced.

Convert sorted linked list to BST (LC 109)? Find midpoint with fast/slow, recurse on both halves. O(n log n) or O(n) with inorder simulation.

#17
Flood Fill
Graph DFS
An image is represented by an m x n integer grid image where image[i][j] represents the pixel value of the image. You are also given three integers sr, sc, and color. You should perform a flood fill on the image starting from the pixel image[sr][sc].
Input
image=[[1,1,1],[1,1,0],[1,0,1]], sr=1, sc=1, color=2
Output
[[2,2,2],[2,2,0],[2,0,1]]
Key insight: DFS from the starting cell. Only fill cells matching the original color. Edge case: if starting cell's color already equals new color, return immediately to avoid infinite loop.
Python 
def floodFill(image, sr, sc, color) -> list[list[int]]:
    original = image[sr][sc]
    if original == color: return image  # avoid infinite loop

    def dfs(r, c):
        if (0 <= r < len(image) and
            0 <= c < len(image[0]) and
            image[r][c] == original):
            image[r][c] = color
            dfs(r+1, c); dfs(r-1, c)
            dfs(r, c+1); dfs(r, c-1)

    dfs(sr, sc)
    return image
Time: O(m×n) Space: O(m×n) — recursion
🔄 Follow-up

Number of Islands (LC 200): Same DFS pattern. Count how many times you start a fresh DFS (each fresh start = one island). Mark visited cells to avoid revisiting.

#18
Binary Search
Binary Search Array
Given an array of integers nums which is sorted in ascending order, and an integer target, write a function to search target in nums. If target exists, then return its index. Otherwise, return -1.
Input
nums = [-1,0,3,5,9,12], target = 9
Output
4 (index of 9)
Key insight: Each iteration halves the search space. Two templates exist — while l <= r (find exact value) vs while l < r (find boundary). Know both cold. Use mid = l + (r - l) // 2 to avoid overflow (matters in Java/C++).
Python 
def search(nums: list[int], target: int) -> int:
    l, r = 0, len(nums) - 1
    while l <= r:
        mid = l + (r - l) // 2   # safe mid
        if   nums[mid] == target: return mid
        elif nums[mid] < target:  l = mid + 1
        else:                     r = mid - 1
    return -1
Time: O(log n) Space: O(1)
🔄 Follow-up

First/last occurrence? When found, don't return immediately. Move r = mid - 1 (for first) or l = mid + 1 (for last) and keep searching. LC 34.

Search in rotated array (LC 33)? Determine which half is sorted, check if target is in that half, then decide which side to discard.

#19
First Bad Version
Binary Search
You are a product manager and currently leading a team to develop a new product. Unfortunately, the latest version of your product fails the quality check. Since each version is developed based on the previous version, all the versions after a bad version are also bad. You are given an API bool isBadVersion(version) which returns whether version is bad.
Input
n=5, bad=4 → isBadVersion: [F,F,F,T,T]
Output
4 (first bad version)
Key insight: Classic "find first true" binary search template. When isBadVersion(mid) is true, mid could be the answer so move right pointer to mid (not mid-1). Loop ends when l == r.
Python 
def firstBadVersion(n: int) -> int:
    l, r = 1, n
    while l < r:               # not <=
        mid = l + (r - l) // 2
        if isBadVersion(mid):
            r = mid            # mid could be answer, keep it
        else:
            l = mid + 1       # mid is good, discard it
    return l                   # l == r == first bad
Time: O(log n) Space: O(1)
🔄 Follow-up

Why while l < r and not l <= r? Because when isBadVersion(mid) is true we set r = mid (not mid-1). If l == r, the loop would never terminate. The l < r guarantees termination.

Given a string s containing just the characters '(', ')', '{', '}', '[' and ']', determine if the input string is valid. An input string is valid if brackets are closed in the correct order and by the same type of brackets.
Input
s = "()[]{}"

True
Key insight: Stack tracks opening brackets. When you see a closing bracket, check if it matches the top of the stack. If stack is empty at the end, valid.
Python 
def isValid(s: str) -> bool:
    stack = []
    mapping = {')': '(', '}': '{', ']': '['}
    for char in s:
        if char in mapping:
            if not stack or stack.pop() != mapping[char]:
                return False
        else:
            stack.append(char)
    return not stack
Time: O(n) Space: O(n)

#20
Implement Queue using Stacks
Stack Queue
Implement a first-in-first-out (FIFO) queue using only two stacks. The implemented queue should support all the functions of a normal queue (push, peek, pop, and empty).
Input
push(1), push(2), pop(), peek()
Output
[1, 2]
Key insight: Use two stacks - one for push operations, one for pop operations. When popping or peeking, if the pop stack is empty, move all elements from push stack to pop stack (reversing order).
Python 
class MyQueue:
    def __init__(self):
        self.stack_in = []  # for push
        self.stack_out = [] # for pop/peek

    def push(self, x: int) -> None:
        self.stack_in.append(x)

    def pop(self) -> int:
        self.peek()  # ensure stack_out has elements
        return self.stack_out.pop()

    def peek(self) -> int:
        if not self.stack_out:
            while self.stack_in:
                self.stack_out.append(self.stack_in.pop())
        return self.stack_out[-1]

    def empty(self) -> bool:
        return not self.stack_in and not self.stack_out
Time: Amortized O(1) Space: O(n)
🔄 Follow-up

Implement Stack using Queues (LC 225): Reverse operation - use queues to implement stack. For pop operation, move all but last element to back queue.

#41
Backspace String Compare
Stack String
Given two strings s and t, return true if they are equal when both are typed into empty text editors. '#' means a backspace character.
Input
s = "ab#c", t = "ad#c"
Output
True (both become "ac")
Key insight: Stack approach: push chars, pop on '#'. O(n) space. But the optimal O(1) space solution scans from right-to-left, skipping characters that would be deleted, using a skip counter.
Python 
# O(n) stack approach (simpler to explain)
def backspaceCompare(s: str, t: str) -> bool:
    def process(string: str) -> str:
        stack = []
        for ch in string:
            if ch == '#':
                if stack: stack.pop()
            else:
                stack.append(ch)
        return ''.join(stack)
    return process(s) == process(t)

# O(1) space — two pointer from right
def backspaceCompare_opt(s: str, t: str) -> bool:
    i, j = len(s) - 1, len(t) - 1
    skip_s = skip_t = 0
    while i >= 0 or j >= 0:
        while i >= 0:
            if s[i] == '#': skip_s += 1; i -= 1
            elif skip_s: skip_s -= 1; i -= 1
            else: break
        while j >= 0:
            if t[j] == '#': skip_t += 1; j -= 1
            elif skip_t: skip_t -= 1; j -= 1
            else: break
        if i >= 0 and j >= 0 and s[i] != t[j]: return False
        if (i >= 0) != (j >= 0): return False
        i -= 1; j -= 1
    return True
Time: O(n+m) Space: O(1) two-ptr / O(n) stack
🔄 Follow-up

Multiple consecutive '#' on empty string? Handled — we only pop if stack is non-empty (or skip only if skip > 0 in the two-pointer version).

Number of 1 Bits
Bit Manip
Write a function that takes the binary representation of a positive integer and returns the number of set bits it has (also known as the Hamming weight).
Input
n = 11 (binary: 00000000000000000000000000001011)
Output
3
Key insight: n & (n-1) clears the lowest set bit. Count how many times you can do this before n reaches 0 — that's the number of 1 bits. Faster than checking every bit individually.
Python 
def hammingWeight(n: int) -> int:
    count = 0
    while n:
        n &= n - 1   # clear lowest set bit
        count += 1
    return count

def hammingWeight_v2(n: int) -> int:
    return bin(n).count('1')

#37
Counting Bits
Bit Manip DP
Given an integer n, return an array ans of length n + 1 such that for each i (0 <= i <= n), ans[i] is the number of 1's in the binary representation of i.
Input
n = 5
Output
[0, 1, 1, 2, 1, 2]
Key insight: dp[i] = dp[i >> 1] + (i & 1). Shifting right drops the last bit (halves the number), and the count of 1s in the half is already computed.
Python 
def countBits(n: int) -> list[int]:
    dp = [0] * (n + 1)
    for i in range(1, n + 1):
        dp[i] = dp[i >> 1] + (i & 1)
    return dp
Time: O(n) Space: O(1)
🔄 Follow-up

Alternative DP relation: dp[i] = dp[i & (i-1)] + 1. i & (i-1) clears the lowest set bit.

#38
Reverse Bits
Bit Manip
Reverse bits of a given 32 bits unsigned integer.
Input
n = 43261596 (00000010100101000001111010011100)
Output
964176192 (00111001011110000010100101000000)
Key insight: Extract LSB with n & 1, shift it to the correct reversed position (31 - i), OR into result. Shift n right by 1 each iteration. 32 iterations total — always O(1).
Python 
def reverseBits(n: int) -> int:
    result = 0
    for i in range(32):
        result = (result << 1) | (n & 1)
        n >>= 1
    return result

# Alternative: shift bit to correct position
def reverseBits_v2(n: int) -> int:
    result = 0
    for i in range(32):
        bit = (n >> i) & 1
        result |= bit << (31 - i)
    return result
Time: O(1) — always 32 iters Space: O(1)
🔄 Follow-up

If called many times? Cache 8-bit reversed values in a lookup table (256 entries). Reverse each byte, combine with shifts. O(1) per call, O(1) setup.

#39
Missing Number
Bit Manip Math
Given an array nums containing n distinct numbers in the range [0, n], return the only number in the range that is missing from the array.
Input
nums = [3, 0, 1]
Output
2
Key insight (Math): Expected sum of [0..n] = n*(n+1)/2. Actual sum = sum(nums). Difference is the missing number. XOR approach also works: XOR all indices and all values — duplicates cancel, leaving the missing number.
Python 
# Math approach — O(1) space
def missingNumber(nums: list[int]) -> int:
    n = len(nums)
    return n * (n + 1) // 2 - sum(nums)

# XOR approach — avoids potential sum overflow (matters in C++)
def missingNumber_xor(nums: list[int]) -> int:
    xor = len(nums)
    for i, n in enumerate(nums):
        xor ^= i ^ n    # each i and n XOR'd; duplicates cancel
    return xor
Time: O(n) Space: O(1)
🔄 Follow-up

Find the duplicate instead (LC 287)? Floyd's cycle detection on the array treated as a linked list — value at index is the "next pointer".

#40
Single Number
Bit Manip
Given a non-empty array of integers nums, every element appears twice except for one. Find that single one.
Input
nums = [4, 1, 2, 1, 2]
Output
4
Key insight: XOR is self-inverse: a ^ a = 0 and a ^ 0 = a. XOR all numbers together — every duplicate cancels itself out, leaving only the single number. O(1) space, O(n) time.
Python 
def singleNumber(nums: list[int]) -> int:
    result = 0
    for n in nums:
        result ^= n
    return result

# One-liner using functools.reduce:
from functools import reduce
from operator  import xor
def singleNumber_v2(nums: list[int]) -> int:
    return reduce(xor, nums)
Time: O(n) Space: O(1)
🔄 Follow-up

Every element appears 3 times except one (LC 137)? Count bits modulo 3 across all numbers — a bit in the answer appears exactly once, all other bits appear 3× and cancel.
